Enhancements

Home page

  • Re-designed the user journey through the service. The user can now expect to enter the service and be immediately shown a list of entities available to manage. This should make it easier and more obvious to the user where to start the journey

Search

  • The Search page has been removed. In Version 1 and Version 2 the user had a separate page (and link in the naviagtion menu) to a page containing a bespoke search function. However due user research leading to a change in the user journey through the servive, the intention with Search going forward is to enable a search/sort on the list of Entities that the user is shown on the Home page. Filtering/sorting and eventual searching of this list is the full search capabilities that were originally
